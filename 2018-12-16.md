6. 递归
```python
def fact(n):
    if n==1:
       return 1
    return n * fact(n - 1)
```
需要注意可能会栈溢出，*理论上，所有的递归函数都可以写成循环的方式，但循环的逻辑不如递归清晰。*

8. 切片
```python
L=['a','b','c','d','e']
L[1:3]
L[:3]
L[:4:2]  '''取0-3，每两个取一个'''
L[::2]   '''取全部，每两个取一个'''
```
会出a,b,c
   a,b,c
   a,c
   a,c,e
